 Store Analytics System with ReID Tracking
An advanced computer vision system for retail analytics that tracks visitors, analyzes demographics, and generates comprehensive store footfall insights using YOLO detection, BoT-SORT tracking with ReID, and ResNet-based classification.

📋 Features
Core Capabilities
Person Detection & Tracking: YOLO11x with BoT-SORT tracker enhanced with Re-Identification (ReID)

Demographic Classification: Multi-head ResNet50 for gender, age, and ethnicity prediction

Footfall Analytics: Entrance, store zone, and exit tracking with pass-by detection

Crowd Optimization: Specially tuned parameters for crowded retail environments

Track Persistence: Classification locking with multi-frame voting for accuracy

Smooth Visualization: Bounding box and label position smoothing to reduce jitter

Comprehensive Analytics: Detection metrics, confidence distributions, and track persistence analysis

Advanced Features
✅ ReID-Enhanced Tracking: Maintains IDs across occlusions and crowded scenes

✅ Ethnicity Fallback: Assigns "unknown" for low-confidence ethnicity predictions

✅ Multi-Frame Voting: Weighted confidence voting for robust classification

✅ Ground Truth Export: Full detection data exported to CSV for analysis

✅ Visual Analytics: Precision-recall curves, confidence distributions, timeline plots
